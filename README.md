# Sentinel Arm - Autonomous Cybersecurity Command Center

## Overview
**Sentinel Arm** is an autonomous cybersecurity command center and threat intelligence engine. It provides state-of-the-art security monitoring, threat detection, and incident response capabilities.

## Features
- 🛡️ Real-time threat detection and monitoring
- 🎯 Advanced threat intelligence gathering
- 🤖 Autonomous command center operations
- 📊 Comprehensive security analytics and reporting
- 🔐 Enterprise-grade security infrastructure

## Technology Stack
- **Frontend**: HTML5, CSS3, JavaScript
- **Styling**: Custom CSS with responsive design
- **Fonts**: Orbitron, Inter, JetBrains Mono
- **Deployment**: GitHub Pages

## Getting Started

### Prerequisites
- Node.js 16 or higher (if building locally)
- Modern web browser (Chrome, Firefox, Safari, Edge)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/THAMARAISELVAM-A/sentinel-arm.git
   cd sentinel-arm
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to the local development URL.
