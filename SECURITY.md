# Security Policy

## Supported Versions

The following versions of AI SOC Sentinel are currently being supported with security updates:
| Version | Supported |
| --- | --- |
| 1.0.x | ✅ |
| < 1.0 | ❌ |

## Reporting a Vulnerability

We take the security of AI SOC Sentinel seriously. If you believe you have found a security vulnerability, please report it to us by following these steps:

1. **Email us**: Please do not report security vulnerabilities via public GitHub issues. Instead, email the maintainers at `security@example.com` (replace with your actual email).
2. **Provide Details**: Include as much information as possible, such as a proof-of-concept, to help us understand and fix the issue.
3. **Be Patient**: We will acknowledge your report within 48 hours and provide a timeline for a fix.

## Our Policy

- Keep information about any vulnerabilities confidential until we have had a chance to fix them.
- We will not pursue legal action against researchers who find and report vulnerabilities in good faith.
- We will give appropriate credit to those who help us improve the security of the project.
